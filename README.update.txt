
Documentation

Requirements: Debian Etch

Update
    
1) Update the system running:
    
    apt-get update
    apt-get dist-upgrade
	
2) Download the last stable version of GNUPanel and unpack on a directory.
	
3) Change to the previous directory.
	
4) Start the script: ./install-debian-dep.sh

5) Start the installation script: ./gnupanel-install.sh
	
5) Execute gnupanel-update.sh
